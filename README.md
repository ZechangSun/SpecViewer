# SpecViewer
A simple visualization tool for astronmers.

## Dependencies
- Python >= 3.7.4
- PyQt5 >= 5.15.4
- pyqtgraph == 0.10.0
- numpy >= 1.19.4

## How to use
```
python main.py
```

You may need to modify `script.py` to adjust the data interface to the GUI.
